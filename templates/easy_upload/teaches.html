Upload Teaches

Please ensure you have uploaded the subjects and users before uploading teaches. Make sure you follow the following data format for uploading. You can only upload CSV files. The fields in [] is optional. Please don't include the brackets while uploading the CSV. Please dont miss any headers or change the position of them.

The CSV should contain the following fields:

teacher subject sem sec [batch] [sub_batch] ug count is_active
8768282 19CS11 1 A Y 60 Y
8768283 19CSL12 1 A A1 B Y 60 N

Please note that the ug field only takes 'Y' (Yes) or N (No). In teacher field the unique ID or the username should be added. Download this csv file for reference. Also, please make sure you don't add batch and subbatch to theory subjects. Contact Admin if you face any problems.
